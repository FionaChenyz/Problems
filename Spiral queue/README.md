     例如：2(1,0),3(1,1),4(0,1),5(1,-1)........

     v=(2t-1)^2 连续奇数(1,3,5,....)的平方个
   3.第4层上某个位置上的数是多少？
     上边：Utop = v+(x+y);
     左边： Uleft= v+(3*x-y);
     下边：Ubottom = v + (-x - 5*y);
     右边：Uright = v+(-7*x+y);
